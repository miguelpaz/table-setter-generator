TableSetterGenerator – Version: 0.1.0

TableSetterGenerator is a rails generator that will add TableSetter functionality to your rails app.

Installation

To install the generator run:

gem install table table_setter_generator

and then generate the required files in your rails application's directory:

script/generate table_setter

Usage

For a detailed walkthrough on how TableSetter works you should consult the TableSetter documentation. In Rails the routes, controller and views are in the usual places. The TableSetter configuration files are in config/tables directory.
